Istruzioni per l'installazione

  • Scarica il file ZIP cliccando sul pulsante sopra
  • Estrai il contenuto in una cartella a tua scelta
  • Segui le istruzioni contenute nel file LEGGIMI.txt
  • Configura le credenziali email e la lista dei volontari
  • Inizia a inviare email automaticamente ai volontari!

Requisiti di Sistema

  • Connessione internet attiva per l'invio delle email
  • Account Gmail con "Password per le app" configurata
  • File Excel (.xlsx o .xls) con i dati dei trasporti
